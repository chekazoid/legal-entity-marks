# Roadmap

Бэклог будущих доработок Legal Entity Marks. Пункты ниже пока не реализованы.

## Фильтр статей с найденными метками в списке записей

На странице `wp-admin/edit.php` добавить фильтр, отбирающий только статьи, где сканер нашёл маркировку (и, по аналогии, статьи с запрещёнными ссылками).

Набросок реализации:

- хук `restrict_manage_posts` рисует `<select>` рядом со штатными фильтрами;
- `pre_get_posts` / `parse_query` вешает `meta_query`:
  - «с маркировкой»: ключ `_lem_matches` (`LEM_META_KEY`), условие `meta_value != ''`;
  - «с запрещёнными ссылками»: ключ `_lem_banned_links` (`LEM_BANNED_LINKS_META_KEY`);
- опционально: колонка в списке записей с числом найденных сущностей и/или ссылок.

Мета уже пишется сканером, данные есть, нужен только UI и запрос.
